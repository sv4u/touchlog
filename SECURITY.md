# Security Policy

## Supported Versions

These versions of `touchlog` are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2.3.x   | :white_check_mark: |
| < 2.3   | :x:                |

## Reporting a Vulnerability

Please send an email to [sasank@vishnubhatlas.net](mailto:sasank@vishnubhatlas.net)
to report a vulnerability.
